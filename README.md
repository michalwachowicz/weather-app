# weather-app

> A simple weather app built with OpenWeatherMapAPI, VanillaJS and SASS

![Project Preview](https://raw.githubusercontent.com/michalwachowicz/weather-app/main/preview.png)

## Design

I designed the entire app before coding it!

👉 [Link to design](https://www.figma.com/file/J91BfTd7sOnkcyjQxriQMA/Weather-App?node-id=3%3A1421 'Design Figma File')

## Built with

- Figma
- HTML
- SASS
- JavaScript
- Webpack
- Babel
- ESLint
- OpenWeatherMapAPI

## Features

- [x] Show current weather
- [x] Show hourly and daily forecast
- [x] Search form
- [x] Units switcher (celsius/fahrenheit)
- [x] Location service to display weather at your location
- [x] Clean design with animations
